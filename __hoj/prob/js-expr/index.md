## 問題
JavaScriptの文`S`が文字列として与えられます。  
`S`がJSの構文として正しければ`Yes`を、正しくなければ`No`を出力しなさい。  

## 入力形式

```
S
```

## 制約
* 1<=|S|<=1000
* SはES5の文として正しい∨SはES5以降の文として正しくない

## 入出力例

### 入力例1
```
1
```

### 出力例1
```
Yes
```

### 入力例2
```
(1+1
```

### 出力例2
```
No
```

### 入力例3
```
console.log( "1" )
```

### 出力例3
```
Yes
```
